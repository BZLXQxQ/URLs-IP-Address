# URLs-IP-Address
一个实现批量URLs解析IP到查询归属地功能的简单小工具。

Requirements：Python3

Usage：

1、urls.txt 中写入目标地址列表

2、命令行运行 python UIA_beta.py

3、输出 URL_IP.txt 和 IP_Address.txt
